## 배열

### 배열이란?

- 같은 변수를 연결하여 만드는 참조형 변수
	- Ex) int[5] arr -> int 변수 5개 모아서 만들어진 int 형 배열변수 arr

- 각각의 값들은 배열의 변호로 직접 접근이 가능하다.
	- Ex) arr[0] = 5 System.out.println(arr[0])

- 배열의 경우에는 그 크기가 선언할 때 정해져야 한다.
	- Ex) int[] arr = new int[]	-> x
			 int[5] arr = new int[] -> o
			 int[] arr = new int[5] -> o
